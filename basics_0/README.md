# Networking Basics #0

This project introduces the fundamental concepts of computer networking.

## Learning Objectives

- Understand the OSI model and its layers.
- Differentiate between LAN, WAN, and the Internet.
- Understand MAC and IP addresses.
- Learn the differences between TCP and UDP.
- Understand common network ports such as:
  - SSH (22)
  - HTTP (80)
  - HTTPS (443)

- Use networking tools such as `netstat` and `ping`.
- Write Bash scripts that interact with networking utilities.

## Files

| File                         | Description                                        |
| ---------------------------- | -------------------------------------------------- |
| 0-OSI_model                  | Answers about the OSI model                        |
| 1-types_of_network           | Answers about LAN, WAN, and Internet               |
| 2-MAC_and_IP_address         | Answers about MAC and IP addresses                 |
| 3-UDP_and_TCP                | Answers about TCP and UDP                          |
| 4-TCP_and_UDP_ports          | Bash script displaying listening TCP and UDP ports |
| 5-is_the_host_on_the_network | Bash script that pings an IP address five times    |

## Requirements

- Ubuntu 22.04 LTS
- Bash scripts must be executable
- Scripts must pass ShellCheck
- First line: `#!/usr/bin/env bash`
- Second line: Comment describing the script
